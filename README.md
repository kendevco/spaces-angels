### Quick Start

```bash
# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local

# Run database migrations
pnpm payload migrate

# Start development server
pnpm dev
```

### Environment Setup

Required environment variables:
```env
DATABASE_URL=postgresql://...
PAYLOAD_SECRET=...
```
